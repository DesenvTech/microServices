## Installation

```bash
$ docker-compose up --build -d
```

## Open Keycloak

[http://localhost:8080/auth](http://localhost:8080/auth).
```bash
username: admin
password: admin
```

## Open Swagger

[http://localhost:3000/api](http://localhost:3000/api).

## Import collection to Postman

Import **postman_collection.json** file to Postman.

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
